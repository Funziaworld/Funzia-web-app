# Funzia World Website

A modern Next.js website for Funzia World - Lagos Bumper Spot.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Responsive design
- Modern UI with Tailwind CSS
- Multiple pages (Home, Gallery, About, FAQ, Contact)
- Booking information
- Service showcases

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
